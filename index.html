Projects I've Worked On

From machine learning to automated bots. A full project I pioneered, designed, and coded will be used in production by a Fortune 15 company (Chevron).

Oilspill Bot

Built an automated process that parses the National Oceanic and Atmoshperic Administration (NOAA) website to extract satellite data of of oil spills. It then filters the data for uniquenss, severity, and proximity to Chevron's pipes and platforms, and finds oilspills that are within proximity of these assets. The process then alerts the control center in order to prevent loss of containment and environmental incidents.

Tools used: Python, Azure, Oracle Databases, Beautiful Soup, Oracle Databases, ArcMap, ArcPy

WildFire Bot

Created a bot that extracts up to the hour wildfire data from a NASA website. It then intelligently merges wildfire points into larger wildfire polygons that represent a single fire and filters out fires that are not severe enough or do not have a high confidence rating. Next, It performs geospatial analysis in order to find only the fires that are within proximity of Chevron pipes and building so that it can alert the control center to ensure the safety of all Chevron employees and products.

Tools used: Python, Azure, Oracle Databases, Beautiful Soup, Oracle Databases, ArcMap, ArcPy

Predicting Trip Purposes (Machine Learning)

Used California Transport data to predict the purpose of a person's trip given spatial, temporal, and personal features (income, education, etc.). I learned to visualize the data set in order to first gain insights into it and fix errors in a dataset with over 42,000 rows and 50 features (columns). I used recursive feature selection to find the best features for prediction and remove features that were not good predictors. I found the best standard models for prediction (Gradient Boosting Tree and Random Forest Classifier) and built my own predictive model, that outperformed the best of these standard models by 4%

Tools used: Python, Pandas, Numpy, Scikit Learn
