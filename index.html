Projects I've Worked On

From machine learning to automated bots. A full project I pioneered, designed, and coded will be used in production by a Fortune 15 company (Chevron).

Oilspill Bot

I built an automated process that parses a NOAA website to extract satellite data of of oil spills. It then filters the data for uniquenss, severity, and proximity to Chevron's pipes and platforms, and finds oilspills that are within proximity of these assets. The process then alerts the control center in order to prevent loss of containment and environmental incidents.

Tools used: Python, Azure, Oracle Databases, Beautiful Soup, Oracle Databases, ArcMap, ArcPy

WildFire Bot

I created a bot that extracts up to the hour wildfire data from a NASA website which then intelligently filters out fires that are not severe enough or do not have a high confidence rating and performs geospatial analysis in order to find only the fires that are within proximity of Chevron pipes and building. It then alerts the control center of the fire. Passing along all the fireâ€™s data and the asset impacted to ensure the safety of all Chevron employees and products.

Tools used: Python, Azure, Oracle Databases, Beautiful Soup, Oracle Databases, ArcMap, ArcPy

Predicting Trip Purposes (Machine Learning)

I used California Transport data to predict the purpose of a person's trip given spatial, temporal, and personal features (income, education, etc.). I learned different ways to visualize the data set in order to first gain insights and fix errors in a dataset with over 42,000 rows and 50 features (columns). In order to used recursive feature selection to find the best features for prediction and remove features that were not good predictors. I found the best standard models for prediction (Gradient Boosting Tree and Random Forest Classifier) and used recursive feature selection optimize the base models further by only including features that served as good predictors. Finally, I built my own predictive model, that outperformed the best of these standard models by 4%

Tools used: Python, Pandas, Numpy, Scikit Learn
